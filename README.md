# QR Share

Легкая Windows-утилита для быстрой передачи файла на телефон или другой компьютер по локальной сети через QR-код.

Идея простая:

1. Нажимаете ПКМ по файлу.
2. Выбираете `QR Share`.
3. Приложение поднимает локальную раздачу и показывает QR-код.
4. Второе устройство сканирует код и скачивает файл.

## Что умеет

- добавляет пункт `QR Share` в контекстное меню файлов
- поднимает локальный HTTP-сервер на свободном порту `8000-8099`
- генерирует QR-код и прямую ссылку на скачивание
- показывает имя файла, размер и путь
- позволяет быстро скопировать ссылку
- позволяет быстро скопировать путь к файлу
- позволяет открыть папку с файлом прямо из окна приложения
- упаковывается в один `exe` и отдельный инсталлятор

## Для кого это

`QR Share` полезен, когда нужно быстро перекинуть файл:

- с ПК на телефон
- с ноутбука на другой компьютер в той же сети
- без мессенджеров, облаков и лишних промежуточных сервисов

## Как это работает

Приложение не загружает файл никуда во внешний интернет.

Оно:

- определяет локальный IP-адрес компьютера
- находит свободный порт
- запускает локальный сервер
- отдает выбранный файл по прямой ссылке
- показывает эту ссылку в виде QR-кода

Файл доступен, пока окно приложения открыто и сервер не остановлен.

## Установка

Самый простой вариант:

1. Откройте готовый установщик `Output/QR_Share_Setup.exe`.
2. Установите программу с правами администратора.
3. После установки пункт `QR Share` появится в контекстном меню файлов.

Что делает установщик:

- копирует `main.exe` в `Program Files`
- добавляет пункт в shell-меню через реестр Windows

## Использование

1. Нажмите правой кнопкой мыши по нужному файлу.
2. Выберите `QR Share`.
3. Дождитесь появления окна с QR-кодом.
4. Подключите телефон или другой компьютер к той же сети.
5. Отсканируйте QR-код и скачайте файл.

## Windows 11

Текущая версия использует классическую shell-интеграцию.

- В Windows 10 пункт открывается в обычном контекстном меню.
- В Windows 11 пункт обычно находится в `Показать дополнительные параметры`.

## Ограничения

- оба устройства должны быть в одной локальной сети
- файл раздается только пока окно приложения открыто
- за один запуск приложение работает с одним выбранным файлом
- firewall Windows может попросить доступ к сети при первом запуске

## Сборка из исходников

### Требования

- Windows 10 или Windows 11
- Python 3.12
- Inno Setup 6

### Установка зависимостей

```powershell
python -m venv venv
.\venv\Scripts\activate
pip install -r requirements.txt
```

### Сборка `exe`

```powershell
.\venv\Scripts\pyinstaller.exe main.spec --noconfirm --clean
```

Результат:

- `dist/main.exe`

### Сборка инсталлятора

```powershell
& "C:\Program Files (x86)\Inno Setup 6\ISCC.exe" setup.iss
```

Результат:

- `Output/QR_Share_Setup.exe`

## Структура проекта

- `main.py` — основная логика приложения и UI
- `main.spec` — конфиг сборки PyInstaller
- `setup.iss` — скрипт сборки инсталлятора Inno Setup
- `requirements.txt` — зависимости Python
